### Start using it now

```bash
# install
npm i --save-dev @nomiclabs/buidler # or yarn add -D @nomiclabs/buidler 

# in a separate terminal
npx ganache-cli

# create a new Buidler project
npx buidler

# run your tests
npx buidler test

# run your custom scripts
npx buidler run scripts/sample-script.js
```
