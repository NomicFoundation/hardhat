## Commands

Install - pnpm install

Build - pnpm build

Test - pnpm test

## Repository layout

v-next/\* – publishable packages

v-next/hardhat - core logic and cli

## Rules

- Always update tests when changing behavior.

## Development workflow

When modifying a package, within the package run:

1. pnpm lint
2. pnpm build
3. pnpm test
