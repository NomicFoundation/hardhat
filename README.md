Hardhat is an Ethereum development environment for professionals. It facilitates performing frequent tasks, such as running tests, automatically checking code for mistakes or interacting with smart contracts.

Built by the [Nomic Foundation](https://nomic.foundation/) for the Ethereum community.

---

**Version compatibility warning:** This branch of the repository only contains package versions that do not work with Hardhat 2 nor 3. Please visit the [main branch](https://github.com/NomicFoundation/hardhat/tree/main) for the latest version of Hardhat.
