Hardhat is an Ethereum development environment for professionals. It helps you perform common tasks, such as running tests, automatically checking code for mistakes, and interacting with smart contracts.

Built by the [Nomic Foundation](https://nomic.foundation/) for the Ethereum community.

---

> 💡 This is the README for Hardhat 3, the new major version of Hardhat. For the previous version (v2), see [this branch](https://github.com/NomicFoundation/hardhat/tree/v2).

---

## Getting started

To install Hardhat and initialize a new project, run the following command in an empty directory:

```bash
npx hardhat --init
```

This command launches an interactive setup process to help you get started.

## Learn more

To learn more about Hardhat, check out the [documentation](https://hardhat.org/docs/).
